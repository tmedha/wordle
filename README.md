# wordle

A Wordle clone — plain HTML/CSS/JS, no build step, no dependencies. Deployable to GitHub Pages as-is.

## Features

- Random 5-letter word each game, no repeats until the whole word list has been used, then it cycles again.
- **Untimed** mode (unlimited time, 6 guesses) and **Timed** mode (3 minutes total, 6 guesses).
- Replay support, with the option to switch modes.
- Progress (in-progress board, per-mode stats, word cycle) persists across page reloads via `localStorage`.

## Running locally

Serve the directory with any static file server (ES modules are blocked under `file://`):

```
python3 -m http.server 8000
```

Then open `http://localhost:8000/`.

## Word database

`data/words.js` (5,735 words) is generated by `tools/generate_words.py` from `words.txt` — the
Stanford GraphBase 5-letter word list, ordered by frequency of use in English. The generator takes
the most frequent 70% as the "common" tier and the least frequent 30% as the "hard" tier (plus a
small blocklist of vulgar/offensive terms). Re-run it any time `words.txt` changes:

```
python3 tools/generate_words.py
```

## Deploying to GitHub Pages

1. Push this repo to GitHub.
2. In repo Settings → Pages, set Source to "Deploy from a branch", Branch `main`, folder `/ (root)`.
3. No build step or GitHub Actions workflow needed — it's static files served as-is.
